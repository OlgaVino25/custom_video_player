# Пользовательский медиаплеер

Простой, но функциональный медиаплеер. Плеер демонстрирует навыки работы с HTML5, CSS3, JavaScript и интеграции сторонних библиотек.

## Функциональность

Плеер предоставляет пользователю все основные функции для комфортного просмотра видео:

- **Воспроизведение и пауза:** Удобные кнопки с иконками из Font Awesome.
- **Индикатор прогресса:** Визуальный прогресс-бар, отображающий текущую позицию в видео.
- **Перемотка:** Возможность перематывать видео, кликая на прогресс-бар.
- **Отображение времени:** Текущее время и общая длительность видео.
- **Контроль громкости:** Включение/выключение звука.
- **Полноэкранный режим:** Просмотр видео на весь экран.

## Технологии

- **Python 3.x**
- **HTML5:** Семантическая разметка и структура плеера.
- **CSS3 (Flexbox):** Стилизация и адаптивная верстка интерфейса.
- **JavaScript:** Логика взаимодействия с элементами управления.
- **Playable.js:** Внешняя библиотека для кроссплатформенной и кроссбраузерной работы с HTML5 video.
- **jQuery:** Используется для упрощения работы с DOM (подключается через CDN).
- **Font Awesome:** Иконки для кнопок управления.
- **livereload:** Сервер для разработки с автоматической перезагрузкой страницы.

## Структура проекта
      
**fonts** - Локальные шрифты (Font Awesome)

**static** - Статические файлы (CSS, JS)
- favicon.ico - иконка сайта
- font-awesome.min.css - стили для иконок

**templates** - HTML шаблоны
- book_template.html - шаблон страницы книги
- template.html - шаблон главной страницы
- player.js - основной JavaScript-код логики плеера. Для работы он требует двух библиотек - jQuery и Playable.

**index.html** - Главный HTML-файл

**serve.py** - Сервер для разработки

**requirements.txt** - Зависимости Python

**README.md** - Документация


## Установка и Запуск

Для запуска у вас уже должен быть установлен `Python 3.x`

1. Клонируйте репозиторий.

2. Установите зависимости командой:

```python
pip install -r requirements.txt
```

3. Запустите сервер разработки:

```python
python serve.py

# Откройте http://127.0.0.1:5500
```

## Особенности реализации

- **Чистая архитектура:** Логика управления плеером вынесена в отдельный файл `player.js`.
- **Кастомный UI:** Интерфейс полностью стилизован под конкретный дизайн, без использования стандартных браузерных контролов для video.
- **Гибкость:** Для создания видео-инстанса используется библиотека Playable, что делает код более переносимым и надежным.
- **Интерактивность:** Прогресс-бар реагирует на клики, реализовано плавное переключение между иконками (Play/Pause, Volume/Mute).
